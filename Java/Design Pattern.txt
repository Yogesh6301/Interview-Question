2. Name the main types of design patterns.

Creational Patterns: Deal with object creation. Examples: Singleton, Factory, Builder, Prototype, Abstract Factory.
Structural Patterns: Deal with object composition and relationships. Examples: Adapter, Decorator, Proxy, Composite, Facade.
Behavioral Patterns: Deal with communication between objects. Examples: Observer, Strategy, Command, Template Method, Iterator, Chain of Responsibility.
